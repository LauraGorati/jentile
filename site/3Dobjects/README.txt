Insert here the models in .glb format linked to their corresponding .txt file for generating the description.

To convert from .obj to .glb, follow thes guide:

1-Install Node.js  https://nodejs.org to get npm
2-from COMMAND PROMPT: npm install -g obj2gltf 
3-from COMMAND PROMPT: obj2gltf -i model_in.obj -o model_out.glb
